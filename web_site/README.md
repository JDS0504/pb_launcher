# PBLauncher Website

Marketing and documentation website for PBLauncher, the open source PocketBase instance manager.

- Production URL: https://pblauncher.ksaucedo.com
- Repository: https://github.com/user0608/pb_launcher

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## SEO

The site includes canonical metadata, Open Graph tags, JSON-LD structured data, `robots.txt`, and `sitemap.xml` for the production domain.
